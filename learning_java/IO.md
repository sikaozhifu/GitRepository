### 四个抽象类

|   输入流    |    输出流    |
| :---------: | :----------: |
| InputStream | OutputStream |
|   Reader    |    Writer    |

_输入和输出的区别：_

两种：文件角度，程序角度。

我们一般的输入、输出都是指从程序角度看

从文件中读数据，程序角度看为输入流，_Reader_

由文件往外写数据，程序角度看为输出流。_Writer_
